# Smart Safety Monitor

A real-time AI-based safety monitoring system built using Python, OpenCV, and YOLOv8.

## Features (Planned)

- Person Detection
- Mobile Phone Detection
- Helmet Detection
- Safety Vest Detection
- Weapon Detection
- Screenshot Capture
- CSV Logging
- Dashboard Analytics

## Tech Stack

- Python
- OpenCV
- YOLOv8
- Ultralytics
- Pandas
- Matplotlib
